A derivative of "Kanban", a task tracking and management system. Expanded on the idea of rule based states and user roles. Was built during a hackathon with a team of four using the MEAN Stack. Idea came from the need for small teams to be able to effeciently track and verify progress on projects.

Denvr

A clone of "Flikr" that allows for users to upload images and sort though the images based on the faces in the image. To accomplish this I came up with a versitile microservice architecture that separated image uploading from the "face-finder" The whole app was orchistrated using docker and was written in TypeScript, JavaScript, and Python; using Amazon S3, RabbitMQ, Redis, and MongoDB. For "face-finder" we used OpenCV.

Radius Chat

Named for the fact that it is a radius based group chat app. The idea is that a user can set the radius in meters from how far they want to recieve messages from. Each message has a location that is associated with it for the server to dicern which users to send that message to. This app was written using the MEAN stack, using Ionic 1.0 instead of Angular.
